## 뭡니까?
저장할 때 마다 reloadSources를 합니다.

## 실행가능하게 하려면
`git clone` 후 `cd my_repo` 후 `pub global activate --source path .` 하면 `dart-reload-on-save`로 실행이 가능해야 합니다.
자세한 것은 [여기](https://dart.dev/tools/pub/cmd/pub-global) 참고
